# BukkitFramework
bukkit web框架

引入方式
```xml
<!--添加Maven仓库-->
<repositories>
    <repository>
        <id>bukkitweb</id>
        <!--仓库暂时只支持IPV6网络-->
        <url>http://dev.mrcsh.cn:8081/repository/virus-cui-release/</url>
    </repository>
</repositories>
<!--Maven依赖-->
<dependencies>
    <dependency>
        <groupId>cn.mrcsh</groupId>
        <artifactId>BukkitWebFramework</artifactId>
        <version>RELEASE</version>
    </dependency>
</dependencies>
```
